# RaspberryPiPicoPacmanAnimation16x16ledmatrix
Micropython Raspberry PI 16x 16 LED WS2812 matrix to animate a ghost sprite

Is used the Neopixel library so you need to install that to the PICO using Thonny's tools > manage packages > then search for neopixel. 

You also need a 16x16 WS2812 matrix with the first led at the bottom right and with the leds arranged in an S formatation. 

Micropython
Raspberry PI Pico
WS2812 
USB Cable that supports data (how many times has this caught be me out!)
some cables to connect the ws2812 to the pico
